What is TableMerger?

TableMerger is a table comparer that allows to unify in a single matrix the information that the features could have in different data sets. Thus, many of the softwares used to make the putative annotations (e.g. MetaboSearch or Ceu Mass Mediator 3.0) only receive a list with the experimental mass of the detected features, and generate a table with the information of the annotated compounds. With TableMerger it is possible to combine in a single matrix the table with the putative annotations and another table with additional information (e.g. peak area, statistical analysis results, RT, among others) of the features.

Therefore, TableMerger receives two user-specified tables as input. The equivalent features of both tables are identified thanks to the experimental mass (Experimental mass) and the retention time (RT [min]). However, if the RT is not contained in any of the input tables, only the experimental mass will be used. The output of this module will be a single matrix with the combined information from the two data sets.

Since the experimental mass of the features can be expressed with different number of decimals in both tables, the user will be able to specify the number of decimal places to which to round.
